# Lottery Web Frontend

Frontend application for the **Lottery Results & Favorite Games** system.

This project provides a modern user interface that allows users to:

- Upload XLSX files containing Lotofácil draw results  
- View imported files  
- Create and manage favorite lottery games  
- Recalculate game statistics  
- Visualize whether a favorite game has already been drawn  
- View the highest number of matches achieved  

---

## Tech Stack

- React  
- Vite  
- TypeScript  
- Material UI (MUI)  
- React Query  
- Axios  

---
